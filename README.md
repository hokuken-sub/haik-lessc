haik LESS compiler
==========
less compiler for haik.

Copyright 2013 by *Hokuken.Inc*


Requirements
-------------

* bash


Installion
-----------

1. clone git repository to your home.
    `% clone git://github.com/toiee/haik-lessc.git ~/.haik-lessc`
2. export PATH
    `% echo 'export PATH=~/.haik-lessc/bin:$PATH' >> ~/.bash_profile`
3. restart shell
    `% source ~/.bash_profile`
4. check haik-lessc
    `% haik-lessc --help`
